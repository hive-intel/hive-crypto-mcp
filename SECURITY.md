# Security Policy

## Reporting a vulnerability

Email **security@hiveintelligence.xyz** with details and reproduction steps.
Please do not open public issues for security reports. We aim to acknowledge
within 2 business days.

## Scope

This repository contains documentation, client configuration, the typed
`client/` SDK, and a Model Context Protocol registry manifest. The Hive MCP
server that powers `https://mcp.hiveintelligence.xyz/mcp` is a managed service;
report server-side issues to the same address.
